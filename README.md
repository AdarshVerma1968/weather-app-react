# Weather App

A React + Vite project that fetches real-time weather data using OpenWeatherMap API.

## Features
- Search weather by city
- Dynamic UI with Material-UI
- Error handling for invalid entries
- Responsive design

## Tech Stack
- React
- Vite
- Material-UI
- OpenWeatherMap API
